# starwars-universe

A React-based web application that explores the Star Wars universe using the public Swapi.Tech API.  
Users can browse characters and films, view detailed information, and navigate seamlessly between sections.

 Features

- Landing page introducing the application
-  Characters page
- Fetches and displays Star Wars characters
- Click a character to view detailed information
- Local search to filter characters by name

- Films page
  - Fetches and displays Star Wars films
  - Click a film to view detailed information
  - Local search to filter films by title

Technologies Used

- React (Vite)
- React Router DOM
- JavaScript (ES6+)
- Tailwind CSS
- SWAPI.Tech (Star Wars API)git
